# Marker Detection App

A React Native application for detecting custom square markers using a mobile camera with region-based processing.

---

## 🚀 Features

- Live camera feed using Vision Camera
- Fixed scan box for guided alignment
- Region-based detection for better accuracy
- Supports solid and mixed-edge markers
- Automatic orientation correction (0°, 90°, 180°, 270°)
- Perspective transformation for accurate extraction
- Outputs normalized 300x300 marker image
- Real-time preview and capture system
